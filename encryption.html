Encryption

Encryption is used to encode information, requiring a key to be read/decoded back to a usable form, preventing a malicious actor from reading the data, even if intercepted. There are two main forms of encryption used in computing:

Symmetric

Image created by MarcT0K, licensed under CC BY-SA 4.0

Symmetric encryption works using a single key to encrypt or decrypt data - Anyone with this key can decrypt the data encrypted with it, or encrypt data to be read with the key.

Asymmetric

Image created by MarcT0K, licensed under CC BY 4.0

Asymmetric encryption works using a public key to encrypt data, and a private key to decrypt this data. This provides more security than symmetric encryption as your public key can be provided to encrypt data for transmission that can only be read with the private key.
